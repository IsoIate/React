## reducer 여러 개 사용하기
- reducer를 여러 개 사용하고 싶으면... 
  - redux state 생성
  - store에 등록 (combineReducers 함수 사용)
  - state를props화 하는 함수 내에 state를 구분하여 작성

```
function state를props화(state){
  console.log(state);
  return {
    state : state.reducer,
    alert열렸니 : state.reducer2 //리듀서2에 있는거 가져오는법
  }
}
export default connect(state를props화)(Cart);
```
