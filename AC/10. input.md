## input
- 사용자 입력값을 받음
- state에 저장하는게 일반적
- input 태그 내에 onChange 이벤트 핸들러를 달아서 사용
- e.target 은 지금 이벤트가 동작하는 HTML 요소를, .value는 input에 입력한 값을 의미함


#### 사용방법
```
function App (){

  let [입력값, 입력값변경] = useState('');

  return (
    <div>
      HTML 잔뜩있는 곳...
      <input onChange={ (e)=>{ 입력값변경(e.target.value) } />   // <- 자바스크립트 문법
    </div>
  )
}
```
